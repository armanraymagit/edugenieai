## Run Locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Set up environment variables:
   ```bash
   cp .env.example .env
   ```
   Then, add your `GEMINI_API_KEY` to the `.env` file.
3. Run the app:
   ```bash
   npm run dev
   ```
